Homepage for Homework 4

Discription

This page consists of an overview of data collected for homework 3 from my Data Visualization class. The home page also includes information about dataset used in the projected visualization.

For this homework assignment I visualized data from DataSF's 311 cases. I choose to select the "Blocked Street or Sidewalk" catagory to visualize the number of cases for the month of Feburary 2020. With this collected data I created a projected visualization in d3.js (version 5).

Data

The data set used was from "DataSF 311 Cases". This dataset includes cases generally associated with a place or thing generally logged by a user regarding their own needs.
The data collected for this assignment ranges only on the month of Feburary 2020.
